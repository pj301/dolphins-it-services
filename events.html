System Actors

  • Manage credentials
Couple/Candidate

Main event participants

  • View guest lists (Wedding only)
  • Access uploaded photos
  • Create photo albums
  • Manage event content

  • View landing page

  • View ceremony details
  • Upload event photos
  • Access photo galleries
  • View event albums
System Process Flow
1. Event Creation & Setup

Organizer creates event details including:

  • Couple/Candidate information
  • Event schedule and venue
  • Photo upload limitations
  • Access credentials
2. QR Code Generation

System automatically generates two QR codes:

  • Landing Page QR: Public access to event details
  • Upload Page QR: Time-restricted photo upload access
3. Guest/Student Interaction
Wedding Landing
Ring Hop Landing
Wedding Upload
Ring Hop Upload

Participants can:

  • Scan QR codes to access event pages
  • Upload photos during event timeframe
  • View shared photo galleries
  • RSVP with guest count (Wedding only)
4. Photo Management & Albums
Gallery & Album Management

Advanced photo management features:

  • Couple/Candidate can curate uploaded photos
  • Create custom albums for display
  • Albums automatically appear on landing page
  • Time-based access control for uploads
